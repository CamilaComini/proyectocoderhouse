Me complace presentar Puntolan, un proyecto en desarrollo que consiste en una tienda en línea de tecnología, creada como parte de un emprendimiento personal.

Este proyecto fue realizado en el marco de un curso de Desarrollo Web impartido por Coderhouse. Durante el curso, adquirí los conocimientos necesarios para construir sitios web completos y funcionales, desde la base con HTML y CSS, hasta aprender a integrar diseño y usabilidad para ofrecer una experiencia de usuario óptima y atractiva.
